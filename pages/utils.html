class utils.fluidState.FluidState(prop1Name=None, prop1Val=None, prop2Name=None, prop2Val=None, fluid=None)[source]¶

Bases: object

FluidState pulls fluid states from coolprop and provides methods for

  • single properties

  • fluid states with all properties.

Units are:

  • P [Pa]

  • T [C]

  • h [J/kg]

  • cp [J/K]

  • rho [kg/m^3]

  • S [J/K]
