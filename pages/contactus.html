keep you updated¶

  • Please consult the list to see whether the identified issues affect your work. If so, please follow the instructions Contributing with genGEO development or request an updated version of the code.

  • If you suspect there is a genGEO Bugs in genGEO, please contact the developers and provide them with a detailed description of the error as well as the conditions and computing environment under which the error occurs as described under Reporting bugs. For a updated list of the bugs and their potiential solution, see known genGEO’s Bugs.
