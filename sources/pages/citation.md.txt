# Citing genGEO

This page lists the references to use if you are using  genGEO for in your publication. For a list of publications that have cited genGEO, please refer to the [List of Publications](./list_of_publications).

## genGEO
For all publications that use gengeo or a genGEO-based application to produce exciting figures, please cite the following.

* Adams, B., Ogland-Hand, J., Bielicki, J. M., Schädle, P., & Saar, M. (2021). Estimating the geothermal electricity generation potential of sedimentary basins using genGEO (the generalizable GEOthermal techno-economic simulator).

:class: dropdown
```

   title = {Estimating the Geothermal Electricity Generation Potential of Sedimentary Basins using genGEO (the generalizable GEOthermal techno-economic simulator},
```
